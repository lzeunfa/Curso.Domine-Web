Converter binário para decimal (inverter)

Binário Decimal

Explicação

Para transformar um número binário em decimal pode fazer o seguinte:

1- Cada dígito do número binário representa uma potência de 2, começando da direita para a esquerda, com o expoente 0.

2- Só calcule onde há 1, pois os 0s não somam nada ao total.

3- Para cada 1, calcule 2^posição e some os resultados.
